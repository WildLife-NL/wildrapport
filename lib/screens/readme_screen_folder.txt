In this folder we ONLY and ONLY keep all the screens that are used in the application.
This means that the files that gather all the widgets and UI components together that forms the final layout to be displayed.

Make sure you include no bussiness logic in this files.
Make sure to respect the SOLID principles. Meaning no direct implementation of any service in this files but 
using its interface instead. In other words respecting the abstraction rules.

Ensure no state management is done in this files. This is done in the managers folder.

Make sure to follow this implementation structure to ensure quality and consistency.
